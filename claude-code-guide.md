# Claude code 사용법

- 골뱅이(@)로 파일 참조 가능
- 프롬프트에 이미지 drag & drop(shift를 눌러서), URL 복붙 가능
- 모드
  - default: 파일 수정 시 항상 유저의 허락을 구함
  - auto-accept edit: 유저 허락 없이 항상 파일을 수정함
  - plan: 파일을 수정하지 않고 제안사항만 제시함
- memory(#)를 사용하여 Claude가 기억해야 할 내용을 저장할 수 있음
- CLAUDE.md 파일에 하기 사항들을 추가하면 좋음
  - 코드 포맷팅 스타일
  - 자주 쓰는 명령어
  - 중요한 파일
  - 선호하는 코딩 패턴
